# Rocky

Software for **Rocky**, an autonomous quadruped robot co-developed with Janne, a fellow student, as an independent robotics project.

## Repository Structure

### ROS2 Packages

#### `gait_controller`

Generates desired foot trajectories for a forward trotting gait.

#### `inverse_kinematics`

Computes the joint angles required to achieve the desired foot positions generated by the gait controller.

#### `angle_mapper`

Transforms kinematic joint angles into physical servo commands, accounting for servo mounting offsets, orientation, and mechanical constraints.

#### `serial_bridge`

Handles serial communication between the ROS2 system and the onboard Arduino controllers, transmitting servo position commands.

---

### Embedded Software

#### `rocky_main`

Arduino firmware responsible for receiving servo angle commands via the serial interface and commanding the servos to the requested positions.

---

## Control Pipeline

```text
gait_controller
        ↓
inverse_kinematics
        ↓
angle_mapper
        ↓
serial_bridge
        ↓
Arduino (rocky_main)
        ↓
Servos
```

The software architecture separates gait generation, kinematics, hardware abstraction, and low-level actuation into independent modules, enabling straightforward testing and future extension with additional gaits, sensors, and autonomous behaviors.
